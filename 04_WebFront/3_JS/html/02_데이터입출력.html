inner HTML

        자바스크립트에서 요소 전체를 읽어들이거나 변경하는 속성
        
        -내용을 읽어올 때 태그 + 속성 + 내용 을 모두 포함함

        -내용을 변경할 때 태그는 HTML 요소로 해석됨

innerText
