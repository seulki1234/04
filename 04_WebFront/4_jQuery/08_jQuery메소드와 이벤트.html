객체(요소) 삽입 메소드

        선택자 요소(A)를 기준으로 삽입 메소드 매개변수에 
        생성된 요소(B) 또는 함수의 리턴값을 추가

        $(A).append(B) : A 요소 내 뒷부분에 B 를 추가(자식)
        $(A).prepend(B) : A 요소 내 앞부분에 B를 추가 (자식)
        $(A).after(B) : A 요소 뒷 부분에 B를 추가(형제)
        $(A).before(B) : A 요소 앞 부분에 B를 추가(형제)

        생성된 요소(B) 를 매개변수로 지정한 선택자 (A) 요소에 추가
        (위와 생성구문의 순서가 반대)

        $(B).appendTo(A) : B를 A의 요소 내 뒷부분에 추가(자식)
        $(B).preppendTo(A) : B를 A의 요소 내 앞부분에 추가(자식)
        $(B).insertAfter(A) : B를 A의 요소 뒤에 추가(형제)
        $(B).insertBefore(A) : B를 A의 요소 앞에 추가(형제)
